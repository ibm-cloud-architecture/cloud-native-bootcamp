# IBM Cloud Native Bootcamp  



## Technologies Covered

#### Cloud Orchestration

- Openshift
- Kubernetes

#### CI/CD

- ArgoCD
- Tekton

## Running Locally

1. Clone this github repo

``` bash
git clone (https://github.com/ibm-cloud-architecture/cloud-native-bootcamp)
```

2. Verify mkdocs and mkdocs material are installed.

``` bash
npm install 
```

3. Build the site

```bash
mkdocs build
```

4. Serve the Site

```bash
mkdocs serve
```
